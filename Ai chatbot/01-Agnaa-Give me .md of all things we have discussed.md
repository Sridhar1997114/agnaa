## 1. The Core Concept

An on-demand platform connecting verified construction workers/labourers
directly to clients (contractors, homeowners, builders) — eliminating the
traditional middleman naka system.

**Three core problems it solves:**
- Workers deliberately slow down work because speed/quality isn't rewarded
- Clients cannot verify worker experience — forced to trust word-of-mouth
- No skill-based pricing — master masons and raw helpers earn the same

- Apps like **Naakaa**, **Digital Labour Chowk (DLC)**, and **Daily Labour App**
  already operate in India confirming real market demand
- **Digital Labour Chowk** was founded 2020, launched March 2023 — barely 3

- India's gig workforce projected to grow to **2.35 crore workers by 2029-30**
- Strategy: **One city, one trade first** — dominate Hyderabad masons before
  expanding

## 3. The Agent-Led Network Model

### The Genius Pivot — Convert Middlemen into Agents

Instead of fighting the munshi/middleman, **appoint them as Agnaa Agents.**

**Why it works:**
- They already have trust with workers at nakas
- They have physical presence every morning
- They know each worker's real skill level from years together

### Agent Structure
- Each agent gets a **unique admin code** — all workers tagged to them permanently
- Agent earns **commission on every booking** from his worker network
- **Hard cap: 50 workers per agent** — maintains quality, prevents monopoly
- Agent handles **ground-level verification** of skills
- Agent gets **mini-dashboard** — worker bookings, ratings, his commissions in real time

### Layered Commission Architecture

| Transaction | Who Gets Paid | Suggested % |
|---|---|---|
| Client pays booking fee | Platform receives | 100% |
| Worker's daily wage | Worker | 75–80% |
| Agent commission | Agent/Middleman | 5–8% per booking |
| Platform net margin | Agnaa | 15–18% |

### Risk Controls
- **GPS-verified job completion** — client confirms on app before payment releases
- **Hard cap at 50 workers per agent** — prevents one agent from controlling supply
- **Direct UPI payment to worker** — not routed through agent, builds worker loyalty
- **Client-side rating overrides** agent-declared skill level over time

## 4. Dynamic Rating-Based Pricing

### Phase 1 — Market Entry
- All workers start at **current local market rate**
  - Mason: ₹800/day | Helper: ₹500/day (Hyderabad)
- No negotiation, transparent pricing

### Phase 2 — Performance Differentiation (After 5+ jobs)

**Rating Formula:**
```

Rating Score = (Avg client rating × 0.7) + (On-time completion rate × 0.2)
+ (Repeat booking rate × 0.1)

```

### Wage Tier System

| Rating | Tier | Daily Rate Premium | Example (Mason ₹800 base) |
|---|---|---|---|
| 4.5–5.0 ⭐ | Elite | +30% | ₹1,040/day |
| 4.0–4.4 ⭐ | Senior | +15% | ₹920/day |
| 3.5–3.9 ⭐ | Standard | Base rate | ₹800/day |
| Below 3.5 ⭐ | Probation | –10% (floored) | ₹720/day |

### Phase 3 — Market Rate Auto-Update
- Base rates indexed to local market quarterly
- Elite workers always shown with rating badge **before booking**

**Why this works:**
- Workers are motivated — clear path to earning more
- Clients self-select quality — pay premium willingly
- Platform grows — same 15–18% take rate on higher transaction values

## 8. Voice-First MVP Strategy

### Why Voice Before App
| Factor | App | AI Voice Agent |
|---|---|---|
| Build time | 6–12 months | 2–4 weeks |
| Build cost | ₹15–40 lakhs | ₹50k–2 lakhs |
| Worker adoption | Low (literacy barrier) | High (just answer phone) |
| Client adoption | Medium | Instant |
| Language support | Needs multilingual UI | Voice handles naturally |
| Maintenance | Constant updates | Minimal |

### How It Works

**Client Booking Flow:**
1. Client calls one Agnaa number
2. AI answers in Telugu/Hindi/English
3. Client says: *"2 masons, 1 helper, tomorrow, Banjara Hills"*
4. AI confirms availability, quotes price, books — no human needed
5. WhatsApp/SMS confirmation sent to client + worker + agent

**Worker Registration Flow:**
1. Agent calls worker once — AI conducts voice interview
2. AI captures: name, Aadhaar, skill type, experience, language
3. Profile auto-created in backend
4. Worker gets Agnaa ID number — their permanent identity

### Recommended Tech Stack

| Tool | Purpose | Cost |
|---|---|---|
| **Bolna AI** | Telugu/Hindi/Hinglish voice agents, India-built | ₹0.80–2/min |
| **Voicory** | Voice + WhatsApp, ₹2,000 free credits | ₹0.80/min |
| **VAPI / Retell AI** | Advanced call routing, developer-friendly | $0.05–0.10/min |
| **WhatsApp Business API** | Post-call confirmation + receipts | ₹0.40/message |

> **Recommended:** Bolna AI — built for India, supports Telugu, handles
> recruitment/booking natively
